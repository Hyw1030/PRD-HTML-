# PRD HTML 知识卡片工具

这是一个纯前端、本地运行的 PRD / Axure HTML 整理工具。

## 它能做什么

- 批量选择 PRD HTML 或 Axure 导出的完整 HTML 文件夹。
- 自动抽取 HTML 页面文字、Axure 页面文字、原型说明和图片数量。
- 输出更适合知识库的“产品知识卡片”。
- 支持下载 Markdown。
- 支持下载 Word（docx）。
- 不上传文件，不调用 API，适合先给团队试用。

## 输出结构

每个需求会整理成：

```md
# 需求名称

## 概括

## 涉及范围

## 原型改动

## 业务规则

## 注意事项

## 关联模块

## 来源信息
```

## 重要说明

本地版不调用 AI，所以它能稳定处理的是 HTML 里已经存在的文字、原型标注、控件文字和页面说明。

如果关键内容只存在于截图里，本地版只能统计图片，不能真正理解图片。要做到更准确的“看图总结原型”，后续需要接入 AI 视觉/OCR。

## 推荐流程

1. 用本工具批量导入历史 PRD HTML。
2. 下载 Markdown 或 Word。
3. 上传到企业知识库或 Obsidian。
4. 后续再接 AI 总结接口，把规则草稿升级为更稳定的产品知识卡片。

## 使用方式

直接打开：

```text
index.html
```

然后选择 PRD / Axure 文件夹即可。
